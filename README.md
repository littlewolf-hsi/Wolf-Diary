# 🐺 WOLF．DIARY 使用說明

歡迎來到WOLF．DIARY。
點擊網址，開始書寫你的故事：https://wolf-diary.vercel.app/

---

## 開始使用

### 註冊與登入

打開網址後會先看到登入畫面。

**第一次來：** 點「註冊」，填入暱稱、電子郵件和密碼（至少 6 個字元），按「建立帳號」。

**之後登入：** 點「登入」，填入電子郵件和密碼。

> 同一組帳號可以在電腦、手機、平板上登入，日記會自動同步，在哪台裝置寫都接得上。

---

## 主畫面

登入後的起點，分成幾個區塊：

**今天** — 顯示你今天寫的日記，點一下就能翻開。還沒寫的話，會看到「今天還沒有日記」的小提示，當作溫柔的提醒。

**統計列** — 目前的連續天數、總篇數、本月篇數，一眼看到自己的累積。

**歷史上的今天** — 把過去同一天寫下的日記翻給你看，適合回頭看看那時候的自己。

**隨機回顧** — 隨機抽一篇舊日記，按「再抽一篇」就換下一篇，常有意外的驚喜。

---

## 新增日記

按右上角的 **✚**（手機）或主畫面右上的「新增日記」（桌機）就能開始。

### 填寫欄位

**日期** — 預設今天，也可以改成過去的日期來補寫。

**標題** — 選填，留白的話會自動取名為「日記」。

**心情** — 先選一個核心心情（共八種：開心、平靜、期待、驚訝、悲傷、疲憊、焦慮、憤怒），右邊會跳出對應的小表情。如果想記得更細一點，下面還能再選第二層細分，例如「開心 → 滿足」、「悲傷 → 孤獨」；不想細分的話，留著「（不細分）」就好。

**提問模式** — 選擇 AI 引導問題的方式：
- **每日一問**：隨機給一個通用的引導問題
- **AI 客製化**：依你的心情和輸入的觸發事件量身生成

選了「AI 客製化」會多出「觸發事件」輸入框，可以填今天的關鍵字或小事（選填），像是「一個人走路回家」、「看了一本好書」。

### 生成引導問題

按「生成引導問題」，AI 會丟一個問題幫你起頭。這個功能需要先到設定頁填入 AI API Key 才能用；沒填也不影響你直接動筆。

### 書寫內容

支援 Markdown 語法：

| 語法 | 效果 |
|---|---|
| `# 標題` | 大標題 |
| `**粗體**` | **粗體** |
| `*斜體*` | *斜體* |
| `> 引言` | 引言區塊 |
| `- 項目` | 無序清單 |

按「閱覽」可以預覽排版後的樣子，按「編輯」切回去繼續寫。右上角會即時顯示字數和行數。

### 儲存

寫完按「儲存日記」，存好後會自動跳到閱覽頁。

> **不用擔心寫到一半弄丟**：切去別的頁面再回來，內容都會留著，接著寫就好。如果是整個關掉視窗再打開，下次會跳出「發現上次未儲存的草稿」，讓你選擇還原或捨棄。

## 閱覽與編輯日記

從主畫面或搜尋頁點任一篇日記，就會進入閱覽頁。

**閱覽模式** — 完整顯示排版後的內容，頂部會標出日期和心情（有選細分的話也會一起顯示）。

**加星號** — 覺得這篇很重要，點右上角的星星把它標起來，星星會亮起；再點一下取消。加了星號的日記，標題前面會有一顆小星，之後也能在搜尋頁用「只看加星號」一次找回來。

**編輯** — 按右上角「✎ 編輯」進入編輯模式，可以修改日期、標題、心情（含細分）和內容。按「儲存變更」後會自動回到閱覽模式。

**刪除** — 按「✕ 刪除」，需要輸入這篇日記的標題確認後才會刪掉。這個動作無法復原，請斟酌。

## 搜尋

點側邊欄的「搜尋」進入搜尋頁。

搜尋頁一開始是空的，會看到一句小提示——選好條件後才會列出結果，畫面比較清爽，也不會跟下面的月曆混在一起。

**關鍵字** — 輸入想找的字，會同時比對標題和內文，按「搜尋」或 Enter 執行。

**日期** — 直接選某一天。

**依心情篩選** — 下拉選某個核心心情，只看那一類的日記。

**只看加星號** — 打開這個開關，就只列出你加過星號的日記。

搜尋結果會出現在搜尋框正下方，不用往下滑。

**月曆** — 在結果下方，標出當月有寫日記的日子，日期底下還有當天的心情表情。點有日記的那天，上方就會篩出那一天；用左右箭頭切換月份。

**清除篩選** — 有套用條件時會出現「清除篩選」按鈕，點一下就清空條件、回到一開始的提示畫面。

---

## 統計

點側邊欄的「統計」，看看自己的書寫軌跡：

- **書寫概覽** — 總篇數、目前連續天數、歷史最高連續天數、最活躍的月份
- **每月篇數** — 長條圖看每個月寫了幾篇
- **最常在哪天寫** — 星期分佈長條圖
- **心情分佈** — 每種心情出現的次數與比例
- **里程碑** — 上一個達成的里程碑，以及距離下一個還有多遠

## 系統設定

點側邊欄的「系統設定」。

### 帳號
- **登出** — 登出目前帳號
- **刪除帳號** — 會刪除你所有的日記與設定並登出，輸入「確認刪除」後執行；此操作無法復原

### 資料
- **匯出日記** — 把所有日記打包成 ZIP 下載，資料夾結構為 `wolf-diary / 年 / 年月 / 日期.md`，是很好的備份
- **匯入日記** — 選先前匯出的 ZIP 匯入，系統會自動略過重複的日記（相同日期與標題的那些）

### AI API
填入 AI 服務的 API 設定，才能使用「生成引導問題」。API Key 只存在你自己的裝置上，不會上傳到伺服器。

### 自訂心情
用 JSON 編輯心情的結構：每個核心心情各有一個 `emoji` 和一組細分 `sub`。例如：
```json
{
  "開心": { "emoji": "😊", "sub": ["滿足", "興奮", "感恩"] },
  "平靜": { "emoji": "😌", "sub": ["放鬆", "安定", "療癒"] },
  "悲傷": { "emoji": "😭", "sub": ["失落", "孤獨", "無力"] }
}
```

### AI 提示詞
自訂 AI 生成引導問題時用的 prompt，可以調整問題的風格和方向。

## 手機加入主畫面

加到主畫面後，開起來就跟一般 App 一樣，也能離線翻看。

**iPhone（Safari）** — 點底部的分享圖示 → 「加入主畫面」

**Android（Chrome）** — 點右上角三點選單 → 「加入主畫面」或「安裝應用程式」

## 常見問題

**日記跨裝置看不到？**
確認兩台裝置登入的是同一組 email 帳號。

**AI 生成問題失敗？**
到系統設定確認 API Base URL、API Key 和 Model 都填對了。

**忘記密碼？**
再跟我說一聲，我幫你重設 🐺

**匯入後日記沒出現？**
可能是日期格式不對，或那篇的日期＋標題和現有日記重複，被自動略過了。
